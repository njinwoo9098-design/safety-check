외도민 안전점검 현장발송앱 PWA v103 - GitHub Pages 업로드용

1. 이 ZIP 파일을 압축 해제합니다.
2. 압축 해제된 폴더 안의 내용물 전체를 GitHub 저장소 첫 화면에 업로드합니다.
3. 저장소 첫 화면에 아래 항목이 바로 보여야 합니다.
   - index.html
   - manifest.webmanifest
   - service-worker.js
   - icons 폴더
   - assets 폴더
4. ZIP 파일 자체를 올리지 마세요.
5. 폴더 자체를 올리지 말고, 폴더 안의 내용물만 올리세요.
6. 업로드 후 휴대폰에서 예전 버전이 보이면 Chrome에서 새로고침 후 홈화면 앱을 완전히 종료했다가 다시 실행하세요.

주의:
- 카카오톡 보내기는 휴대폰 공유창에서 카카오톡을 선택한 뒤 최종 발송은 사용자가 직접 선택합니다.
- 이메일 보내기는 네이버메일/G메일 메일쓰기 화면을 열고 받는사람·제목·본문을 준비합니다.
- 브라우저 보안상 네이버메일/G메일 웹 작성창에 파일을 자동 첨부하는 것은 제한되므로, 앱이 복사해 준 첨부파일 목록을 보고 안전점검표 PDF와 자격증명 서류를 직접 첨부해 주세요.
