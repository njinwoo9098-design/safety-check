외도민 안전점검 현장발송앱 PWA v124-iPad - GitHub Pages 업로드용

기준:
- v114 안정본을 기준으로 다시 제작했습니다.
- 안드로이드/데스크탑용 기존 'PDF 저장 / 인쇄' 기능은 그대로 유지했습니다.

v124-iPad 반영:
- '아이패드 PDF 인쇄' 버튼을 추가했습니다.
- 아이패드 버튼은 Safari 인쇄 시 짝수장 빈 페이지가 생기는 문제를 줄이기 위해 별도 인쇄 CSS를 적용합니다.
- iPad 전용 인쇄 모드에서는 페이지 고정 높이/overflow를 완화하고, 사진표와 이미지가 숨지 않도록 보정했습니다.
- 사진이 없는 4페이지는 기존처럼 제외됩니다.
- 표 크기, 칸 크기, 행 높이, 열 너비는 기존 v114 기준을 유지했습니다.

권장 사용:
- 데스크탑/안드로이드: 기존 'PDF 저장 / 인쇄'
- 아이패드: 새로 추가한 '아이패드 PDF 인쇄'

업로드:
1. ZIP 압축 해제
2. 내용물 전체를 GitHub 저장소 첫 화면에 덮어쓰기 업로드
3. 변경 내용 커밋
4. GitHub Pages 배포 완료 후 새로고침
5. 아이패드 홈화면 앱이 예전 버전을 계속 보이면 홈화면 앱 삭제 후 Safari에서 다시 홈화면 추가
