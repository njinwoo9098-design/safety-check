외도민 안전점검 현장발송앱 PWA v120 - GitHub Pages 업로드용

v120 반영:
- '안전점검표 인쇄' 버튼을 다시 이미지 저장 방식으로 변경했습니다.
- 안전점검표는 PDF가 아니라 이미지 파일로만 저장됩니다.
- 안드로이드는 기본적으로 JPG로 저장됩니다.
- 아이패드/iPhone은 안정성을 위해 PNG로 준비되며, 미리보기/공유창에서 '파일에 저장' 또는 '이미지 저장'을 사용할 수 있습니다.
- '완성된 안전점검표선택'은 JPG/PNG 이미지 파일만 선택할 수 있습니다.
- 카카오톡/문자메세지/이메일 발송 기능은 선택한 이미지 파일을 첨부하는 방식으로 유지했습니다.
- 표 크기, 칸 크기, 행 높이, 열 너비는 변경하지 않았습니다.

업로드:
1. ZIP 압축 해제
2. 압축 해제한 내용물 전체를 GitHub 저장소 첫 화면에 덮어쓰기 업로드
3. 변경 내용 커밋
4. GitHub Pages 배포 완료 후 앱 새로고침
5. 휴대폰/아이패드 홈화면 앱이 예전 버전을 계속 보이면 홈화면 앱 삭제 후 다시 추가
