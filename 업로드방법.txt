외도민 안전점검 현장발송앱 PWA v130-iPad - GitHub Pages 업로드용

기준:
- v114 안정본 계열 유지
- 데스크탑/안드로이드용 기존 'PDF 저장 / 인쇄' 유지
- 아이패드 전용 '아이패드 PDF 인쇄' 유지

v130-iPad 핵심 수정:
- 사진이 좌측/하단으로 붙는 문제를 해결하기 위해 사진을 인쇄 직전에 캔버스로 다시 만듭니다.
- 캔버스 안에서 흰 배경을 먼저 깔고, 사방 3mm에 해당하는 여백을 둔 뒤 사진을 가운데 cover 방식으로 그립니다.
- 따라서 iPad Safari가 object-fit을 잘못 인쇄해도 사진 자체에 사방 여백이 들어가도록 했습니다.
- photo-box의 padding은 0으로 바꾸고, 3mm 여백은 사진 캔버스 내부에서 처리합니다.
- 빈 페이지를 줄이기 위해 page-break-after를 제거하고, 두 번째 페이지부터 page-break-before만 적용했습니다.
- 마지막 페이지 뒤에는 강제 페이지 나눔을 적용하지 않습니다.
- 4페이지는 사진이 있을 때만 인쇄문서에 포함됩니다.

사용:
- 데스크탑/안드로이드: PDF 저장 / 인쇄
- 아이패드: 아이패드 PDF 인쇄

업로드 후 아이패드 홈화면 앱은 삭제 후 Safari에서 새로고침하고 다시 홈화면에 추가하세요.
