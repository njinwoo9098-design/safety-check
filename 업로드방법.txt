v132-iPad 수정사항:
- 다른 화면/표/사진 배치 기능은 변경하지 않았습니다.
- 아이패드 PDF 인쇄 버튼만 직접 PDF 생성 방식으로 변경했습니다.
- 브라우저 인쇄 바닥글을 사용하지 않아 GitHub 주소와 작성날짜가 PDF에 들어가지 않습니다.
- 각 페이지 하단 중앙에는 페이지번호(예: 1 / 5)만 표시됩니다.
- 사진이 없는 4페이지는 기존처럼 제외됩니다.
- 생성된 PDF가 새 화면에 열리면 공유 버튼에서 프린트를 선택하면 됩니다.
