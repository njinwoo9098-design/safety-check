외도민 안전점검 현장발송앱 PWA v106 - GitHub Pages 업로드용

1. 이 ZIP 파일을 압축 해제합니다.
2. 압축 해제된 폴더 안의 내용물 전체를 GitHub 저장소 첫 화면에 업로드합니다.
3. 저장소 첫 화면에 아래 항목이 바로 보여야 합니다.
   - index.html
   - manifest.webmanifest
   - service-worker.js
   - icons 폴더
   - assets 폴더
4. ZIP 파일 자체를 올리지 마세요.
5. 폴더 자체를 올리지 말고, 폴더 안의 내용물만 올리세요.
6. 업로드 후 휴대폰에서 예전 버전이 보이면 Chrome에서 새로고침 후 홈화면 앱을 완전히 종료했다가 다시 실행하세요.

v106 반영:
- 개설신고확인증과 건축사자격증 사본 하단에 빨간 박스 문구를 자동 생성합니다.
- 문구: 이 문서는 외국인관광 도시민박업 신청용으로 [신청주소]에 한정하여 제출함
- [신청주소]는 현장발송패널의 신청주소 또는 1장 주소란 값으로 자동 반영됩니다.
