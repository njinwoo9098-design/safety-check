외도민 안전점검 현장발송앱 PWA v127-iPad - GitHub Pages 업로드용

기준:
- v114 안정본 계열 유지
- 데스크탑/안드로이드용 기존 'PDF 저장 / 인쇄' 유지
- 아이패드 전용 '아이패드 PDF 인쇄' 유지

v127-iPad 반영:
- 공통 인쇄 방향을 A4 세로(portrait)로 강제했습니다.
- 아이패드 PDF 인쇄 시 전체 페이지가 A4에 더 꽉 차게 나오도록 여백을 줄였습니다.
- 아이패드 사진 3·4페이지가 잘리지 않도록 사진표 높이와 페이지 여백을 재조정했습니다.
- 아이패드 인쇄 후에도 사진이 사진틀 안쪽 약 3mm 여백만 두고 꽉 차게 나오도록 img 출력 방식을 absolute가 아닌 block 방식으로 변경했습니다.
- 사진이 있는 4페이지는 인쇄 직전에 강제로 표시합니다.
- 사진 페이지 뒤에 빈 페이지가 따라 나오는 현상을 줄이도록 page-break를 보정했습니다.

사용:
- 데스크탑/안드로이드: PDF 저장 / 인쇄
- 아이패드: 아이패드 PDF 인쇄

업로드 후 아이패드에서 기존 홈화면 앱은 삭제 후 Safari에서 새로고침하고 다시 홈화면에 추가하세요.
