외도민 안전점검 현장발송앱 PWA v129-iPad - GitHub Pages 업로드용

기준:
- v114 안정본 계열 유지
- 데스크탑/안드로이드용 기존 'PDF 저장 / 인쇄' 유지
- 아이패드 전용 '아이패드 PDF 인쇄' 유지

v129-iPad 핵심 수정:
- 아이패드 인쇄 미리보기에서 빈 페이지가 보이는 문제를 줄이기 위해 iframe 크기를 1px가 아닌 실제 A4 크기(210mm×297mm)로 변경했습니다.
- 아이패드 전용 인쇄문서 body에서 기존 ipad-pdf-mode 클래스를 제거하여 누적된 iPad 보정 CSS 충돌을 차단했습니다.
- 마지막 출력 페이지에는 last-print-page 클래스를 부여하여 마지막 뒤 빈 페이지가 생기지 않도록 했습니다.
- 사진 DOM을 아이패드 인쇄용으로 새로 구성하고, 사진 이미지에 inline !important 스타일을 직접 부여했습니다.
- 사진은 사진틀 안쪽 3mm 여백을 두고 object-fit:cover 방식으로 꽉 차게 출력되도록 재보정했습니다.
- 사진표 높이는 A4 한 페이지 안에 들어가도록 282mm로 조정했습니다.
- 4페이지는 사진이 있을 때만 인쇄문서에 포함됩니다.

사용:
- 데스크탑/안드로이드: PDF 저장 / 인쇄
- 아이패드: 아이패드 PDF 인쇄
