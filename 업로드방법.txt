외도민 안전점검 현장발송앱 PWA v114 - GitHub Pages 업로드용

v114 반영:
- PDF 저장/인쇄 시 파일명이 신청주소 기준으로 적용됩니다.
- 파일명 형식: [신청주소] 건축물 안전점검표
- 예: 서울시 마포구 환일길 5-1 아현동 건축물 안전점검표
- 파일명에 사용할 수 없는 특수문자는 자동으로 공백 처리합니다.
- 표 크기, 칸 크기, 행 높이, 열 너비는 변경하지 않았습니다.

업로드:
1. ZIP 압축 해제
2. 압축 해제한 내용물 전체를 GitHub 저장소 첫 화면에 덮어쓰기 업로드
3. 변경 내용 커밋
4. GitHub Pages 배포 완료 후 앱 새로고침
