외도민 안전점검 현장발송앱 PWA v125-iPad - GitHub Pages 업로드용

기준:
- v114 안정본 계열을 유지합니다.
- 안드로이드/데스크탑용 기존 'PDF 저장 / 인쇄' 기능은 그대로 유지했습니다.

v125-iPad 반영:
- 아이패드 PDF 인쇄 시 사진이 빠지는 문제를 보정했습니다.
- 아이패드 PDF 인쇄 시 4페이지 사진표가 안 나오는 문제를 보정했습니다.
- page4에 사진이 있으면 인쇄 직전에 skip-print를 해제하고 강제로 표시합니다.
- 아이패드 인쇄 전에 사진 이미지 로딩을 최대 3초 대기한 뒤 인쇄창을 엽니다.
- 아이패드 전용 인쇄 모드에서 사진표와 사진 이미지를 Safari가 인쇄하기 쉬운 구조로 보정했습니다.
- 표 크기, 칸 크기, 행 높이, 열 너비는 기존 v114 기준을 유지했습니다.

권장 사용:
- 데스크탑/안드로이드: 기존 'PDF 저장 / 인쇄'
- 아이패드: '아이패드 PDF 인쇄'

업로드:
1. ZIP 압축 해제
2. 내용물 전체를 GitHub 저장소 첫 화면에 덮어쓰기 업로드
3. 변경 내용 커밋
4. GitHub Pages 배포 완료 후 새로고침
5. 아이패드 홈화면 앱이 예전 버전을 계속 보이면 홈화면 앱 삭제 후 Safari에서 다시 홈화면 추가
