외도민 안전점검 현장발송앱 PWA v118 - GitHub Pages 업로드용

v118 긴급수정:
- v117에서 일부 발송/인쇄 기능이 멈추던 스크립트 오류를 수정했습니다.
- 'PDF 저장 / 인쇄' 버튼은 화면에서 제거했습니다.
- '이미지 저장' 버튼명은 '안전점검표 인쇄'로 유지했습니다.
- '완성된 안전점검표 PDF 선택(필수)' 문구는 '완성된 안전점검표선택'으로 유지했습니다.
- 완성된 안전점검표선택은 PDF/JPG/PNG/WEBP 파일과 여러 장 선택을 지원합니다.
- 카카오톡/문자메세지/이메일 전송 기능이 다시 동작하도록 복구했습니다.
- 표 크기, 칸 크기, 행 높이, 열 너비는 변경하지 않았습니다.

업로드:
1. ZIP 압축 해제
2. 압축 해제한 내용물 전체를 GitHub 저장소 첫 화면에 덮어쓰기 업로드
3. 변경 내용 커밋
4. GitHub Pages 배포 완료 후 앱 새로고침
5. 휴대폰 홈화면 앱이 예전 버전을 계속 보이면 홈화면 앱 삭제 후 다시 추가
