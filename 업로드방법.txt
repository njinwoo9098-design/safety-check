외도민 안전점검 현장발송앱 PWA v117 - GitHub Pages 업로드용

v117 반영:
- 'PDF 저장 / 인쇄' 버튼을 화면에서 제거했습니다.
- '이미지 저장' 버튼명을 '안전점검표 인쇄'로 변경했습니다.
- '완성된 안전점검표 PDF 선택(필수)' 문구를 '완성된 안전점검표선택'으로 변경했습니다.
- 완성된 안전점검표선택은 PDF뿐 아니라 JPG/PNG/WEBP 이미지도 선택할 수 있고, 여러 장 선택이 가능합니다.
- 표 크기, 칸 크기, 행 높이, 열 너비는 변경하지 않았습니다.

업로드:
1. ZIP 압축 해제
2. 압축 해제한 내용물 전체를 GitHub 저장소 첫 화면에 덮어쓰기 업로드
3. 변경 내용 커밋
4. GitHub Pages 배포 완료 후 앱 새로고침
