외도민 안전점검 현장발송앱 PWA v128-iPad - GitHub Pages 업로드용

기준:
- v114 안정본 계열 유지
- 데스크탑/안드로이드용 기존 'PDF 저장 / 인쇄' 유지
- 아이패드 전용 '아이패드 PDF 인쇄' 유지

v128-iPad 핵심 수정:
- 아이패드 인쇄 버튼은 현재 화면을 그대로 인쇄하지 않고, 별도의 아이패드 전용 인쇄문서(iframe)를 만들어 출력합니다.
- 안드로이드 인쇄 결과를 기준으로 A4 세로 페이지, 페이지 크기, 사진표, 사진 맞춤을 다시 구성했습니다.
- iPad Safari가 기존 화면을 자동 축소하면서 여백이 커지는 문제를 줄이기 위해, 아이패드 전용 인쇄문서는 A4 210×296mm 기준으로 생성됩니다.
- 사진은 사진틀 안쪽 약 3mm 여백을 두고 object-fit:cover 방식으로 꽉 차게 출력됩니다.
- 사진이 있는 4페이지는 인쇄문서에 포함하고, 사진이 없으면 제외합니다.
- 빈 페이지를 줄이기 위해 인쇄문서에는 실제 출력할 페이지 1~5만 순서대로 복제합니다.

사용:
- 데스크탑/안드로이드: PDF 저장 / 인쇄
- 아이패드: 아이패드 PDF 인쇄

업로드 후 아이패드 홈화면 앱은 삭제 후 Safari에서 새로고침하고 다시 홈화면에 추가하세요.
